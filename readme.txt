Để chạy project, hãy làm theo các bước sau:
	B1: Bật cmd
	B2: Gõ lệnh npm i
	B3: Gõ lệnh npm start. Project sẽ chạy với port 3000

Vui lòng chạy trước project-server trước khi chạy project-client
